What I do

I'm a passionate and motivated aspiring Full-Stack Developer, currently building strong foundations in both frontend and backend development. I’ve been actively working on hands-on projects, attending structured training, and continuously sharpening my coding and problem-solving skills.
Below is a quick overview of my main technical abilities and tools I work with. Curious about my journey? Explore my online resume and project portfolio to learn more..

JavaScript

As a core web technology, JavaScript helps me bring interactivity and dynamic behavior to web pages. From manipulating the DOM to handling asynchronous requests with fetch and async/await, I use it to enhance user experience. I also apply ES6+ features to write cleaner, modern code and integrate JavaScript with APIs and UI frameworks for responsive applications.

Angular, React

Both Angular and React play key roles in my frontend development. Angular supports me in building large-scale applications with a structured, component-based approach and two-way data binding. React allows me to create dynamic and reusable UI components, using hooks and the virtual DOM to manage state efficiently. These frameworks help me develop responsive, modern interfaces with a seamless user experience.

Node.js

With Node.js, I develop fast and scalable server-side applications using JavaScript. It allows me to handle backend logic, work with file systems, and create RESTful APIs. I also use Express.js to simplify routing and middleware, and integrate Node.js with databases and frontend frameworks for full-stack application development.

Java

Java enables me to build reliable and scalable applications by applying object-oriented principles. I’ve used it extensively for backend development, creating reusable and clean code across various projects. Development tools like IntelliJ and build tools like Maven streamline my workflow. On an advanced level, I work with frameworks like Spring and Hibernate, and apply multithreading techniques to improve performance and efficiency.

Spring Framework

Spring provides a comprehensive foundation for building Java-based backend systems. I leverage Spring Boot for rapid development and Spring MVC for structured API design. In more complex projects, I implement Spring Data JPA for efficient database interactions and use Spring Security for robust authentication and authorization.

Git

Version control is essential in my development workflow, and Git allows me to track changes, collaborate in teams, and maintain code history effectively. I work with branches, handle merges, and resolve conflicts to keep the workflow smooth. For more advanced tasks, I manage pull requests, review code, and integrate Git with platforms like GitHub for CI/CD automation.

HTML & CSS

Creating clean, accessible web layouts starts with solid HTML structure, and CSS brings those layouts to life with styling and design. I implement responsive design using Flexbox, Grid, and media queries to ensure usability across devices. For more advanced styling, I work with animations, transitions, and preprocessors like SASS to create polished user interfaces.

MySQL

MySQL supports the backend of my applications by managing structured data through relational databases. I design tables, write efficient queries, and maintain data integrity through constraints and relationships. On an advanced level, I work with stored procedures, optimize queries with indexing, and integrate MySQL with backend technologies like Spring Boot and Node.js.
